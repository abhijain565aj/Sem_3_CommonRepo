CS231 Lite EndSem – 30 marks

Instructions

  • You are given an outline of the problems a few days ahead of time

  • You can seek help to solve these problems, but you have to solve them yourself during the exam duration

  • Open one-A4-sheet handwritten notes: during the exam, you can carry one A4 sheet of handwritten notes, which you have to submit to the instructor after the exam

  • These notes should NOT include the solution to the exam problem itself, but can contain any other notes

Mutual Recursion in MIPS ISA – 10 marks

  • You are given C code in file mutual-recursion.c

  • Convert this to MIPS32 assembly code in file mutual-recursion.s – specifically, write code for functions f, h, g, and main

  • You MUST follow the MIPS32 register and calling conventions

  • In the actual exam, you may be given a slightly different g and main function, and asked specific questions based on the code you write

Pipelining, Loop Unrolling – 20 marks

  • You are given C code in file product.c

  • Convert this to MIPS64 assembly code in product.s , to be run in winmips64 – you do not have to do the final printing of the product in the assembly code

  • Try to write your code such that it has only one branch instruction executed per loop, in the main loop

  • Check the correctness of your MIPS64 code by compiling and running the C code

  • Identify various stalls as you run the MIPS64 code – structural, data hazard stalls, control stalls

  • Also identify various instances of data forwarding

  • Now, unroll the loop so that the loop variable increments by 2 in each iteration – in file product2.s

  • Identify instances of anti-dependence after unrolling, and remove these anti-dependences by using additional registers as necessary

  • In the main exam, you will be asked various questions on:

    • the pipelined execution

    • the different kinds of stalls while executing product.s and product2.s

    • performance comparison of with vs without loop unrolling

    • possible code scheduling opportunities after unrolling
